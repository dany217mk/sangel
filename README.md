# Sangel - официальный сайт (v2)

Этот репозиторий содержит вторую версию сайта проекта **Sangel** - сервиса экстренной помощи и агрегатора ГБР частных охранных предприятий.  
В отличие от первой версии, текущая версия разработана по профессиональному дизайну из Figma и ориентирована на реальных пользователей, инвесторов и партнёров.

---

## О проекте

Веб-сайт служит презентационной площадкой для:
- мобильного приложения Sangel  
- умного устройства (кулон / браслет) для вызова помощи  
- информации о сервисе, миссии и технологии  
- демонстрации концепта партнёрам и инвесторам  

Основная цель сайта - формирование доверия и демонстрация продукта на уровне современного коммерческого проекта.
